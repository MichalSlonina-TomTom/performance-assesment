# Performance Assessment

This repository contains the framework and resources for TomTom's performance assessment process.

## Overview

TomTom's performance management framework empowers every TomTom'er to take the lead in their career development. Performance is evaluated across three equal pillars, guided by our core principles.

## Structure

- **[evaluation_framework/](evaluation_framework/)** - The complete performance evaluation framework
  - [Core Principles](evaluation_framework/principles.md) - Fundamental principles that guide all TomTom'ers
  - [Performance Pillars](evaluation_framework/README.md) - The three pillars of performance assessment
    - [Leadership](evaluation_framework/pillars/leadership.md)
    - [Capabilities and Development](evaluation_framework/pillars/capabilities_and_development.md)
    - [Impact](evaluation_framework/pillars/impact.md)

- **goals/** - Performance goals and objectives
- **self_evaluations/** - Self-assessment materials

## Getting Started

1. Review the [core principles](evaluation_framework/principles.md)
2. Understand the [three pillars](evaluation_framework/README.md) of performance
3. Set your goals in the `goals/` directory
4. Complete self-evaluations in the `self_evaluations/` directory
